1. 프로그램 설명
WAC는 Windows OS에서 수집 가능한 아티팩트들을 수집하는 프로그램입니다. 
이 프로그램을 통해 사용자는 다양한 수집 도구가 없어도 여러 아티팩트를 수집할 수 있습니다.
이 프로그램에서는 파일의 시간값과 로컬 검사를 통한 가독성에 도움이 되는 기능이 포함되어 있습니다.

----------------------------------------------------------------------------

2. 시스템 요구 사항
운영 체제: Windows 10 이상
 - 프로그램 자체는 하위 버전에서도 동작하지만, 아티팩트의 경로가 다를 수 있습니다.
저장 공간: 수집한 파일의 크기가 클 수 있습니다. 적당한 공간의 저장 장치를 사용하세요.
메모리: 이 프로그램 동작에 2GB 이상 할당이 가능한 환경이면 원활하게 동작할 수 있습니다.

----------------------------------------------------------------------------

3. 프로그램 실행 방법
설치 후 WAC.exe 아이콘을 더블 클릭하여 실행합니다.

----------------------------------------------------------------------------

4. 주요 기능
기능 1: 윈도우에 존재할 수 있는 모든 아티팩트에 대해 수집할 수 있습니다.
기능 2: 이 프로그램을 실행한 PC에 저장된 아티팩트를 검사하여, 수집 가능한 아티팩트만 볼 수 있습니다.
기능 3: 아티팩트의 시간을 기준으로 검색할 수 있습니다.
	- 이때의 시간은 UTC+00로 저장돼 있으므로, 비교에 유의해야 합니다.

----------------------------------------------------------------------------

5. 사용법
WAC.exe의 기본 사용법은 다음과 같습니다:

기본 사용법:
프로그램을 실행한 후, 우측의 리스트에서 수집하고자 하는 아티팩트를 선택하고 저장할 경로와 Contanier 를 지정하고 '수집하기' 버튼을 누르면 수집이 진행됩니다.
'로컬 아티팩트 검사' 버튼을 클릭하면 현재 PC에 존재하는 아티팩트와 아티팩트의 시간값을 검사합니다.
'로컬 아티팩트 검사'가 진행되면 '현재 수집 가능한 아티팩트' 옵션 사용이 가능합니다.
'현재 수집 가능한 아티팩트'를 체크하면 현재 PC에 존재하는 아티팩트만 리스트에 보여집니다.
'현재 수집 가능한 아티팩트'가 체크되면 '시간 옵션' 사용이 가능합니다.
'시간 옵션'으로 원하는 시간값을 지정하면 해당 범위에 맞는 아티팩트만 리스트에 보여집니다.
